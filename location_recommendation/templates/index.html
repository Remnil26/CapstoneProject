Why LBRS? New Established Region

Location Based

Recomendation

System For

Business





This project aims at recommending suitable

location where newly or pre established

businesses can start and bloom.

By anticipating factors such as industry, competition,

frequencies and their average user ratings.

Using the K-means clustering

algorithm to propose

locations based on the

industry type and average

customer ranking.

Using Correlation analysis

between various sectors

to suggest locations

using a ranking algorithm.

Predicting suitable business

that one may open in a

particular region using

machine learning techniques.

Get recommendations in

one click

Just select the industry type and get real time

geo coordinates on the map.
